# 에이닷 비즈 RPG — CLAUDE.md v9

## 프로젝트 개요
게임명: 신입사원 도토리의 첫 출근
플랫폼: 웹 브라우저 (index.html 단일 파일 실행)
플레이타임: 약 8~10분
특이사항: 무음 (사무실 환경 고려, 모든 피드백은 시각 이펙트로만)

## 기술 스택
- Phaser 3 (lib/phaser.min.js 로컬 참조)
- Vanilla JS (빌드툴 없음, ES6 모듈)
- 맵: AI 생성 배경 이미지 사용 (assets/maps/office_map.png)
  충돌 영역은 코드에서 투명 Rectangle로 별도 처리
- 단일 index.html 진입점
- 대사 관리: 구글시트 → CSV → scripts.js 변환

## 화면 & 맵 스펙

- 카메라: 전체 화면 고정 (스크롤 없음)

## 충돌 처리 방식
- 배경 이미지는 단순 표시용 (충돌 없음)
- 충돌 영역은 코드에서 투명 Rectangle로 처리
- 구역 경계 벽, 가구, 우하단 검정 영역 모두 코드로 막기
- 우하단 검정 영역 (x:1100-1280, y:500-720) → 진입 불가 처리

## 진행 잠금 규칙
- 게임 시작: 로비 + 내 자리 + 휴게실 개방 / 자료실·회의실 잠김
- 스테이지 1 클리어 → 자료실 잠금 해제, 박주임 ❗ 등장
- 스테이지 2 클리어 → 회의실 잠금 해제, 최과장 ❗ 등장
- 스테이지 3 클리어 → 정선배 ❗ 재등장 / 김대리 ❗ 등장
- 엔딩: 정선배에게 말 걸면 시작

## NPC 마커 시스템
- ❗ 노란색 바운스: 말 걸 수 있음
- ❓ 파란색 고정: 대화 진행 중
- ✅ 초록색 고정: 대화 완료
- 없음: 접근 불가 또는 아직 등장 전
- ✦ 반짝임: 조사 가능 오브젝트 (이스터에그)
- 모든 대화 가능 NPC는 항상 ❗ 표시, 별도 힌트 대사 없음

## 씬 구조
- BootScene:   에셋 프리로드 전용
- GameScene:   메인 게임 (배경맵 + 플레이어 + NPC)
- UIScene:     HUD 오버레이 (항상 GameScene 위에 실행)
- EndingScene: 엔딩 연출 (스킬카드 합체 + 마스터 뱃지)

## 시스템 파일
- src/systems/DialogSystem.js:  대화창, 선택지, 타이핑 연출
- src/systems/MissionSystem.js: 미션 상태 관리, 잠금 해제 트리거
- src/systems/EffectSystem.js:  파티클, 쉐이크, 스킬카드 UI

## 데이터 파일
- src/data/scripts.js: 모든 대사, 선택지, 정오답 데이터
- 절대 GameScene이나 DialogSystem 안에 대사 하드코딩 금지
- 이미지 슬롯 트리거도 여기서 관리 (showScreen: 'screen_1')

## 대화창 UI 스펙
```
┌─────────────────────────────────────────┐
│ 🧑‍💼  정선배                              │  상단: 이모지 + 이름 (금색 #FFD700)
│                                         │
│  대사 내용이 한 글자씩 타이핑됨 ▌       │  중단: 타이핑 연출 + ▌ 커서 깜빡임
│                                         │
│  ┌─────────────────────────────────┐    │  하단A: 선택지 버튼 (세로 배치)
│  │ A. 선택지 내용                  │    │        호버 시 테두리 금색 (#FFD700)
│  └─────────────────────────────────┘    │
│                  클릭 또는 SPACE로 진행 ▶│  하단B: 선택지 없을 때만 표시
└─────────────────────────────────────────┘
```
- 배경: rgba(0,0,0,0.5) 반투명 검정
- 테두리: 1px solid rgba(255,255,255,0.15)
- NPC 이름: #FFD700 금색
- 전체 대화창 클릭 = 대사 진행
- 선택지 버튼 클릭은 stopPropagation
- 스킵: 클릭 or 스페이스바 → 전체 텍스트 즉시 표시
- 도토리 독백: 이름 없이 이탤릭체

## 엔딩 스킬카드 연출 순서
1. 정선배 엔딩 인트로 대사 완료
2. 스킬카드 3장 순서대로 등장 (좌→중→우 슬라이드인)
3. 3장이 중앙으로 모이며 합체
4. 에이닷 비즈 마스터 뱃지로 변환 + 골드 파티클 버스트
5. 엔딩 후속 대사 진행
6. 🎉 첫 출근 완료! + [다시 하기] 버튼

## 이펙트 (무음)
- 대사 타이핑: 글자마다 작은 바운스
- 미션 클리어: 화면 가장자리 별 파티클 버스트
- 스킬카드 획득: 카드 슬라이드 등장 + 골드 테두리 반짝임
- 오답 선택: 화면 살짝 흔들림 (쉐이크)
- 정답 선택: 초록 반짝임 + 체크 이펙트

## 조작
- 방향키 / WASD: 도토리 이동
- 스페이스바: NPC 대화 / 오브젝트 조사 / 대사 넘기기
- 숫자키 or 클릭: 선택지 선택

## 코딩 컨벤션
- 모든 함수/변수에 한글 주석 필수
- 상수는 파일 상단 CONFIG 객체로 관리
- 좌표, 프레임 번호 하드코딩 금지
- 대사 데이터는 scripts.js에서만 관리, 절대 하드코딩 금지
